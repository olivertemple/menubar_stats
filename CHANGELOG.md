# Changelog

All notable changes to MenuBarStats will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-15

### Added
- Initial release of MenuBarStats
- CPU monitoring with overall and per-core usage statistics
- Memory monitoring with detailed RAM usage breakdown
- Storage monitoring showing disk usage and available space
- Network monitoring with upload/download speeds
- Network information display (IP address and MAC address)
- Temperature monitoring (CPU and GPU, when available)
- Port scanning with ability to view and kill processes
- Configurable menu bar display (1-2 stats)
- Comprehensive settings panel
  - Choose menu bar primary and secondary stats
  - Toggle individual stats in detail view
  - Adjust refresh interval (0.5-5 seconds)
  - Launch at login option
- Native macOS menu bar integration
- SwiftUI-based user interface
- Persistent user settings
- Background operation (no dock icon)
- Universal binary support (Intel and Apple Silicon)

### Documentation
- Comprehensive README.md
- Detailed BUILDING.md with build instructions
- Complete USAGE.md user guide
- CONTRIBUTING.md guidelines for contributors
- Build script (build.sh) for command-line building
